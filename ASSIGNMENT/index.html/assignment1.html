Basics of HTML

HTML is a markup language that uses tags to define elements in a document. These elements are the building blocks of a web page, including headings, paragraphs, images, links, forms, tables, and more.

HTML Structure: 1. HTML Document: An HTML document is the basic building block of a web page. 2. HTML Elements: HTML elements are represented by tags, which are surrounded by angle brackets (< and >). 3. HTML Tags: HTML tags are used to define the structure and content of a web page. 4. HTML Attributes: HTML attributes provide additional information about an element.



HTML Attributes:

1. id: Specifies a unique identifier for an element. 2. class: Specifies a class for an element. 3. style: Specifies inline styles for an element. 4. href: Specifies the URL of a link. 5. src: Specifies the URL of an image or other media.
